### Dictionary Operations Demo:

```
Develop a project that includes one module script 'mymodule.py' and 'dictionary.py'

* mymodule.py should have the following functions:
- read a value for any purpose
- generate a sample list from first to last of size n (first, last, n are from STDIN)
- generate a a dynamic dictionary of students with random key-{3 digits} and value-{name} from STDIN
- sort and print the dictionary
- update a student name by key
- delete a student by key

* dictionary.py should have a function:
- access all mymodule.py functions and execute them

```
