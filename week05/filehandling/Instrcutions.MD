### File handling Demo:

```
Develop a project that includes one module script 'mapping.py' and 'main.py'

* mapping.py should have the following functions:
- read a value for any purpose
- read a csv file
- convert csv to a list of dictionaries
- convert the list data to JSON and save it to JSON file
- read data from json file 
- convert json object to string and print it out

* main.py should have the following functions:
- extract data from csv and save it to json
- print the data from a json file
- enable a switching option for users to print data or save

```
